# MermaidSketch Privacy Policy

Last updated: May 2026

## Data Collection
MermaidSketch does not collect personal data.

## AI Feature
When using AI generation, your text prompt 
is sent to Google Gemini API using YOUR OWN 
API key. We do not store or process this data.

## Local Storage
All diagrams are stored locally on your device.
We do not have access to your diagrams.
